# BrickBreakerPlus — 项目计划

## 1. 技术方案
- **类型**：纯前端网页版，浏览器直接运行
- **技术栈**：HTML5 + CSS3 + Vanilla JavaScript（Canvas 渲染）
- **无需后端**，零依赖

## 2. 界面结构

```
┌─────────────────────────────────┐
│         开始界面 (start)          │
│     标题 + 最高分 + 开始按钮       │
├─────────────────────────────────┤
│         游戏界面 (playing)        │
│   顶部HUD: 分数 | 关卡 | 生命数    │
│   中部: 砖块区 (Canvas)          │
│   底部: 挡板 + 球                │
├─────────────────────────────────┤
│         暂停界面 (paused)         │
│      覆盖层: 暂停中 + 继续按钮     │
├─────────────────────────────────┤
│       失败结算界面 (gameover)     │
│    本次得分 + 最高分 + 重新开始     │
└─────────────────────────────────┘
```

## 3. 功能模块与实现顺序

### Phase 1 — 核心游戏循环
| 步骤 | 功能 | 验证方式 |
|------|------|---------|
| 1.1 | Canvas 渲染 + 游戏主循环 (requestAnimationFrame) | 页面显示画布 |
| 1.2 | 挡板：鼠标/键盘左右控制，限制边界 | 挡板跟随光标移动 |
| 1.3 | 球：物理运动、与墙壁/挡板碰撞反弹 | 球在画布内正确反弹 |
| 1.4 | 砖块：布局生成、球与砖块碰撞检测+消除 | 砖块被击中后消失 |
| 1.5 | 计分：每消除一个砖块 +10 分，HUD 实时更新 | 分数变化正确 |
| 1.6 | 生命值：球落到底部扣 1 条命，重置球位置 | 命数减少，球重置 |
| 1.7 | 游戏结束：0 命时显示结算界面 | 游戏结束画面正常 |

### Phase 2 — 界面与体验
| 步骤 | 功能 | 验证方式 |
|------|------|---------|
| 2.1 | 开始界面：标题 + 最高分 + 开始按钮 | 打开页面看到开始界面 |
| 2.2 | 暂停功能：按 P/ESC 暂停/继续 | 暂停后游戏停止 |
| 2.3 | 游戏结束界面：显示本次得分、最高分、重新开始 | 游戏结束后正确显示 |
| 2.4 | 最高分持久化：localStorage 读写 | 刷新页面后最高分保留 |
| 2.5 | 视觉风格：渐变背景、圆角砖块、阴影效果、动画 | 界面美观精致 |

### Phase 3 — 关卡与道具
| 步骤 | 功能 | 验证方式 |
|------|------|---------|
| 3.1 | 多关卡：不同砖块布局配置，通关后进入下一关 | 通关后新布局出现 |
| 3.2 | 道具掉落：部分砖块被消除时掉落道具 | 道具向下移动 |
| 3.3 | 道具 — 加宽挡板（持续 10 秒） | 挡板变宽后恢复 |
| 3.4 | 道具 — 多球（分裂为 3 球） | 多个球同时运动 |
| 3.5 | 道具 — 穿透球（穿过砖块不反弹） | 球穿过砖块不反弹 |

### Phase 4 — 音效与打磨
| 步骤 | 功能 | 验证方式 |
|------|------|---------|
| 4.1 | 砖块消除音效 | 消除时播放 |
| 4.2 | 挡板/墙壁碰撞音效 | 碰撞时播放 |
| 4.3 | 生命值减少 + 游戏结束音效 | 触发时播放 |
| 4.4 | 道具拾取音效 | 拾取时播放 |
| 4.5 | 响应式适配 | 不同屏幕尺寸可用 |

## 4. 文件结构

```
brick-breaker/
├── index.html         # 主页面，包含所有 UI 元素
├── style.css          # 样式：渐变、圆角、阴影
├── js/
│   ├── game.js        # 主游戏循环、状态管理
│   ├── ball.js        # 球：运动、碰撞逻辑
│   ├── paddle.js      # 挡板：控制、绘制
│   ├── brick.js       # 砖块：布局、碰撞检测
│   ├── powerup.js     # 道具：掉落、效果
│   ├── particle.js    # 粒子特效（可选美化）
│   ├── audio.js       # 音效管理
│   └── storage.js     # localStorage 最高分读写
└── assets/
    └── sfx/           # 音效文件（.wav 或 .mp3）
```
